    # new Rank()

    Crea una carta de rango
    Source:
    • src/Rank.js, line 84
    Example 
    const rank = new canvacard.Rank()
            .setAvatar(img)
            .setCurrentXP(203)
            .setRequiredXP(500)
            .setStatus("dnd")
            .setProgressBar(["#FF0000", "#0000FF"], "GRADIENT")
            .setUsername("SrGobi")
            .setDiscriminator("5100");
        
        rank.build()
            .then(data => {
                canvacard.write(data, "RankCard.png");
            })
